feng3d
======

风的flash3d游戏引擎

feng3d开始于2014年3月14日

制作这个引擎的最主要的目的是为了掌握3d引擎技术，精通away3d。制作过程参考away3d引擎，难免会有与away3d相似的地方，或者会成为away3d的feng版本，也或者会成为与away3d有极大差别的引擎。

目前这个阶段就是抄袭away3d，先想办法站到away3d这个巨人的肩膀上去。

现在只是个开始，需要一两年甚至几年的努力吧。

 
**master目前为开发分支**

**release为稳定分支，能够保证正常运行**
